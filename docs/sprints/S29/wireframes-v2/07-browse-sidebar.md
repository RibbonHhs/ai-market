## 状态色

| 状态 | 表现 |
|------|------|
| 一级节点 hover | bg `--bg-tertiary` + 文字 `--text-primary` |
| 一级节点 active | bg `--primary-bg` + 文字 `--primary` + 左侧 2px primary |
| 二级节点 hover | bg `--bg-tertiary` |
| 二级节点 active | 文字 `--primary` + 左侧 1px primary |
| 展开/折叠 | 箭头旋转 90° / 0° 200ms ease-out |
| 计数 badge | bg-tertiary + 圆角 4px + 12px text |

## 自检

- [ ] 12 用途色条在展开时与 chip 颜色一致
- [ ] 移动端折叠为顶部下拉
