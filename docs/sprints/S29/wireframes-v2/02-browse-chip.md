# 02 — BrowseView 顶部 Chip 流 — 暗色 wireframe

> **S27 错误**：13 chip 在暗色下仍是白底（漏改）
> **v2 修复**：每个 chip 直接绑 `--chip-X-bg` / `--chip-X-fg`，不依赖全局 `--usage-bg` 反相

## 暗色配色

| 元素 | token | 实色 |
|------|-------|------|
| 容器背景 | `--bg-primary` | `#0d0d0f` |
| Chip 容器 | `--bg-secondary` | `#161618` |
| Chip 默认 bg | `--bg-tertiary` | `#1f1f23` |
| Chip 默认 fg | `--text-secondary` | `rgba(255,255,255,0.68)` |
| Chip hover bg | `--bg-tertiary` → `--bg-elevated` | `#26262c` |
| Chip 选中 bg | `--primary-bg` | `rgba(167,139,250,0.16)` |
| Chip 选中 fg | `--primary` | `#a78bfa` |
| Chip 选中边框 | `--primary-border` | `rgba(167,139,250,0.32)` |
| 12 用途 chip | 见 dark-tokens-v2 §3 | 16% alpha + 浅色 fg |

## 状态色

| 状态 | 表现 |
|------|------|
| Chip hover | bg `--bg-tertiary` → `#26262c` + 1px 边框 `--border-strong` |
| Chip 选中 | bg `--primary-bg` + fg `--primary` + 边框 `--primary-border` 1px |
| Chip focus | 2px 焦点环 `--primary-border` |
| 排序按钮 hover | bg `--bg-tertiary` |
