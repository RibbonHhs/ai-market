## 暗色配色

| 元素 | token | 实色 |
|------|-------|------|
| 区域标题 | `--text-primary` | `rgba(255,255,255,0.92)` |
| 区域副标题 | `--text-secondary` | `rgba(255,255,255,0.68)` |
| "查看更多"链接 | `--primary` | `#a78bfa` |
| SkillCard | 见 03-skill-card | |
| 排名徽章 TOP 1/2/3 | `--warning` / `--bg-tertiary` / `--bg-tertiary` | 金 / 银 / 铜效果用文字色区分 |
| 数字徽章（如收藏数 999+） | `--primary-bg` + `--primary` | |
