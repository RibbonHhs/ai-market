## 状态色

| 状态 | 表现 |
|------|------|
| Tab hover | 文字 text-secondary → text-primary |
| Tab 选中 | 文字 primary + 底部 2px primary |
| 列表行 hover | bg-tertiary |
| 编辑按钮 | bg-tertiary + text-primary + 边框 border |
| 删除按钮 hover | bg danger-bg + 文字 danger |
| 头像 hover | 边框 primary-border + scale(1.02) |

## 自检

- [ ] 头部封面与主背景区分明显（elevated 比 primary 浅 4-11%）
- [ ] 当前 Tab 一眼能识别
- [ ] 危险操作（删除）用 danger 语义色，不用灰色伪装
