# 09 — HomeStats（数字卡片） — 暗色 wireframe

> **S27 错误**：数字卡片在暗色下未适配
> **v2 修复**：用 `--bg-secondary` 作为卡片底，数字用 `--text-primary` 加大字号

## 暗色配色

| 元素 | token | 实色 |
|------|-------|------|
| 卡片 bg | `--bg-secondary` | `#161618` |
| 卡片边框 | `--border` | `rgba(255,255,255,0.08)` |
| 卡片 hover | bg-tertiary + 边框 primary-border | |
| 数字（大） | `--text-primary` | `rgba(255,255,255,0.95)` 32px / 600 weight |
| 标签（小） | `--text-secondary` | `rgba(255,255,255,0.68)` 14px |
| 图标 | `--primary` | `#a78bfa` |
| 趋势上升 | `--success` | `#34d399` |
| 趋势下降 | `--danger` | `#f87171` |

## 状态色

| 状态 | 表现 |
|------|------|
| 卡片 hover | bg-tertiary + border-strong + 阴影 shadow-sm |
| 趋势上升 | 文字 `--success` + ↑ 图标 |
| 趋势下降 | 文字 `--danger` + ↓ 图标 |
| 趋势平 | 文字 `--text-tertiary` + — 图标 |

## 自检

- [ ] 4 个数字一眼能扫读（32px / 600 weight / tabular-nums）
- [ ] 趋势色对比度 ≥ 4.5:1（success / danger 在 dark 上 ≥ 7:1）
- [ ] 图标用 SVG 不 emoji
