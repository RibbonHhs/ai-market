## 状态色

| 状态 | 表现 |
|------|------|
| Tab hover | 文字 text-tertiary → text-secondary |
| Tab 选中 | 文字 text-primary + 底部 2px `--primary` |
| 代码块 | bg-elevated + 顶部 1px 边框 `--border` |
| 收藏按钮激活 | bg `--primary-bg` + 文字 `--primary` + 边框 `--primary-border` |
| 评分 hover | 星形 scale(1.1) + 颜色 `--warning` |
