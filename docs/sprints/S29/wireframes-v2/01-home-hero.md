# 01 — HomeHero（首页双 tab + 搜索卡） — 暗色 wireframe

> **S27 错误**：标题/小字/tab 文字在紫色渐变背景上对比度差（被吃）
> **v2 修复**：渐变 overlay 加 `rgba(13,13,15,0.6)` 半透黑底 + 文字用 `var(--text-primary)` on overlay

## 暗色配色

| 元素 | token | 实色 |
|------|-------|------|
| 渐变背景 | `--bg-primary` + alpha | `linear-gradient(135deg, #0d0d0f 0%, #1a1530 50%, #0d0d0f 100%)` |
| 渐变 overlay | `rgba(13,13,15,0.6)` | 半透黑底，确保文字 ≥ 7:1 |
| 主标题 h1 | `--text-primary` | `rgba(255,255,255,0.95)` |
| 副标题 | `--text-secondary` | `rgba(255,255,255,0.72)` |
| Tab 文字（未选） | `--text-tertiary` | `rgba(255,255,255,0.50)` |
| Tab 文字（选中） | `--text-primary` | `rgba(255,255,255,0.95)` + 下划线 `--primary` |
| Tab 下划线 | `--primary` | `#a78bfa` |
| 搜索卡 bg | `--bg-elevated` | `#26262c` |
| 搜索卡边框 | `--border-strong` | `rgba(255,255,255,0.16)` |
| 搜索输入文字 | `--text-primary` | `rgba(255,255,255,0.92)` |
| 搜索占位 | `--text-tertiary` | `rgba(255,255,255,0.42)` |

## 状态色

| 状态 | 表现 |
|------|------|
| 搜索卡 hover | 边框 `--border-strong` → `--primary` |
| 搜索卡 focus | 边框 `--primary` + 2px 焦点环 `rgba(167,139,250,0.32)` |
| 主按钮 hover | bg `#a78bfa` → `#c4b5fd` + scale(1.02) |
| Tab hover（未选） | 文字 `--text-tertiary` → `--text-secondary` |

## 自检

- [ ] 主标题在渐变 + overlay 后对比度 ≥ 12:1（AAA）
- [ ] Tab 文字在 overlay 后对比度 ≥ 4.5:1
- [ ] 搜索卡与渐变背景有清晰边界（bg-elevated 比 primary 浅 11%）
