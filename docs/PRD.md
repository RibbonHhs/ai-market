# SkillsMap 产品需求文档 (PRD)

> 版本 v1.0 · 2026-06 · SkillsMap Team

## 1. 产品定位

**SkillsMap** 是一个面向 Claude Skills 生态的**浏览、检索、评价与一站式管理平台**。它对标 skillsmp.com，但更注重**中文社区 + 后台管理 + 可扩展**。

## 2. 目标用户

| 角色 | 画像 | 主要诉求 |
|---|---|---|
| **AI 开发者** | 用 Claude Code / Cursor 写代码的全栈/前端 | 快速发现 Skills、查看文档、复制安装命令 |
| **Skill 创作者** | 想发布/分享自己写的 Skill | 后台上传、管理版本、查看下载量 |
| **运营 / Admin** | 平台运营方 | 审核 Skill、维护分类标签、管理用户 |

## 3. 范围

### 3.1 用户端（公开）

- ✅ 首页：Hero 搜索 + 精选/最新/热门 + 分类入口
- ✅ 浏览页：分页 + 关键词搜索 + 分类筛选 + 排序（最新/安装/评分/浏览）
- ✅ 详情页：基础信息 + SKILL.md 渲染 + 评分列表 + 收藏 + 写评价
- ✅ 分类页：分类网格 + 分类内浏览
- ✅ 登录 / 注册（JWT）
- ✅ 个人中心：我的收藏 / 我的评价

### 3.2 管理端（ADMIN 角色）

- ✅ Dashboard：总览统计 + 来源分布
- ✅ Skill 管理：列表 + 搜索 + 上下架 + 编辑 + 删除 + 从本地扫描导入
- ✅ 分类管理：CRUD
- ✅ 标签管理：列表 + 删除
- ✅ 用户管理：列表 + 角色变更 + 启/禁

## 5. User Story

### US-1 浏览 Skills
**作为** AI 开发者  
**我希望** 在首页看到精选 Skills 并能通过搜索/分类快速找到想要的  
**以便于** 提高工作效率

**验收**：
- 首页 hero 区有搜索框
- 至少 3 类 Skills 板块（精选/最新/热门）
- 点 skill 卡片跳详情页
- 关键词搜索跳到 `/browse?keyword=xxx`

### US-2 查看 Skill 详情
**作为** AI 开发者  
**我希望** 查看 Skill 的完整 SKILL.md 文档、安装命令、用户评价  
**以便于** 决定是否安装

**验收**：
- 详情页显示 name / version / license / author / description
- SKILL.md 正文完整渲染（Markdown + 代码高亮）
- 显示安装命令（一键复制）
- 显示评分列表（5⭐ + 评论 + 时间）

### US-3 评分与收藏
**作为** 注册用户  
**我希望** 对 Skill 评分/写评论/收藏  
**以便于** 表达喜好并跟踪自己关心的 Skills

**验收**：
- 未登录提示登录
- 评分 1-5 星 + 评论
- 一用户一 Skill 只能有一条评分（更新覆盖）
- 收藏可切换

### US-4 后台管理
**作为** 平台运营  
**我希望** 在后台增删改查 Skill、分类、用户  
**以便于** 维护内容生态

**验收**：
- admin/admin123 可登录后台
- 后台 dashboard 显示总览统计
- Skill 列表支持状态过滤、上下架、删除
- 分类管理可增删改
- 用户管理可改角色、启/禁

## 6. 验收冒烟用例

| # | 场景 | 步骤 | 预期 |
|---|---|---|---|
| 1 | 浏览首页 | 打开 / | 看到 hero + 精选/最新/热门三组 |
| 2 | 关键词搜索 | /browse 搜 "claude" | 返回相关 skill 列表 |
| 3 | 详情查看 | 点任意 skill | Markdown 渲染 + 评分列表 |
| 4 | 用户注册 | /register 注册 test1/123456 | 注册成功，自动登录 |
| 5 | 提交评分 | 详情页 5⭐+评论 | 提交后详情页评分更新 |
| 6 | 收藏 | 详情页点收藏 | /me 可见 |
| 7 | 后台登录 | /admin admin/admin123 | 进 Dashboard |
| 8 | 新建 Skill | 后台表单填写 | /browse 立即可见 |
| 9 | 上架/下架 | 后台操作 | 列表过滤 |
| 10 | 错误路径 | 未登录访问 /admin | 跳 /login |
